Team Name: Anything Works
Theme: Deep Sea

Description:
Beyond Twilight is a 2D pixel-oriented simulation that, given the theme of deep sea, encourages players to explore a given area of the sea, layer by layer. This is a high resolution data visualization using a 50x50 cell structure which contains 4x4 sub cells making a 200x200 actual grid for the player to explore.

Data Set Files Used: 
Coral, hazards, currents, poi, life, resources.  

Dependency for python 3: pygame library.

Installation command: pip install pygame.

To run project, execute code via the "python" command using the main.py file.

Run command: python main.py.
